URL Parser for C++

This is an url parser for C++.

Functionality:

  • URL parsing
  • URL encoding/decoding
  • query parsing
  • path parsing

Usage:

    ...
    string s = "http://www.google.co.kr/path1/path2/file.html";
    EdUrlParser* url = EdUrlParser::parseUrl(s);
    ...

Refer to example for detail information.
